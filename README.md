# Fighter
## 游戏内容概括
玩家和敌人各拥有一架战斗机。战斗机有很多装备槽。装备槽中可以装备装备。游戏的核心玩法是战斗的时候这些装备每隔一段时间会发动其效果，战斗是自动进行的。玩家要做的仅仅是组合搭配这些装备。

## 装备设计
### 说明
每个装备都应该有以下信息：
1. 名字（就是它叫什么）
2. 类别（一定是武器、防护、模块、战术中的一个，武器即攻击用的装备， 
防护即防御性的装备，模块即辅助性装备，战术指战斗策略）
3. 描述（这个装备的效果是什么）
4. 强度（用于计算装备效果的数值，这个数值可能会用不到）
5. 价格（价格）
6. CD（每隔多少秒触发效果）

- 三年式机枪
    - 武器
    - 大口径机枪，每次攻击使其他武器类别的装备迟滞0.5s
    - 强度：30
    - 价格：80
    - CD：3.5
